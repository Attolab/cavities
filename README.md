# Cavities

This repository contains all the codes used to evaluate non-planar cavity geometries, specifically the ones pertaining to this project. It is divided into the following sections:

1. 1D Circularity plot for varying dephasing:                                              **_1Ddephasing.py_**
2. 2D Circularity plot for varying dephasing and a varying geometric parameter:              **_2Ddephasing_geometry.py_**
3. 1D AOI and Rotation Angle plots for varying geometric parameter:                            **_Rotation_AOI.py_**                  
4. A place to store the parameters for your most used cavity geometries:                     **_input.py_**

SImulating/visualising to check coupling into the cavity eigenmodes as a function of input polarization:
1. Input S amplitude, P amplitude, dephasing variable:                **_polarizationsim.py_**
2. QWP angle variable:             **_qwpsim.py_**

- Text file with some geometric variables for the cavities:    **_commonplot.txt_**
